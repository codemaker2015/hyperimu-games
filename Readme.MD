# HyperIMU Games

# Installation and Running

1. Create a shortcut of server.js

2. Append "node " text infront of the target text in the shortcut

3. Double click on server.js shortcut and type the client ip (mobile ip)

4. Enter the system ip in the hyperIMU and enable accelerometer from the sensor list

5. Start hyperIMU

6. Double click on index.html from Games folder
